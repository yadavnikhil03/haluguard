# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in HaluGuard, please report it privately by emailing the maintainers at **nikhil181245@gmail.com**.

Please do not open public issues for security vulnerabilities.

## What to expect

You will receive a response within 72 hours. Once the issue is confirmed, we will work on a fix and publish it as soon as possible.

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |
